# 物联网运营平台

基于Vue 3 + Vite构建的现代化物联网运营管理平台

## 功能特性

- 设备管理：监控和管理IoT设备
- 数据分析：对IoT设备产生的数据进行可视化和分析
- 告警管理：实时监控设备状态，及时发现异常
- 用户管理：管理平台用户和权限
- 系统设置：配置系统参数和功能

## 技术栈

- 前端框架：Vue 3
- 构建工具：Vite
- 路由管理：Vue Router
- 状态管理：Pinia
- UI组件库：Element Plus
- 图表库：ECharts
- HTTP请求：Axios

## 开发指南

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
